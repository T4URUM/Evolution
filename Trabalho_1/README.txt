	Esse jogo foi criado em minha primeira matéria de programação. O objetivo era recriar o "Guitar hero", utilizando a linguagem C
e uma biblioteca gráfica chamada Allegro. O arquivo fora baseado em um exemplo do professor,, o qual continha apenas as rotinas de ini-
cialização e de fechamento. Após algumas semanas de desenvolvimento esse fora o resultado! Divirta-se!!!

	Para executar o jogo, rode o "ghero-0".
